Pinch-to-Zoom

Sie können in den Graphen zoomen, indem Sie mit zwei Fingern eine Bewegung auseinander oder zusammen machen.

Scrollen

Sie können seitlich scrollen, indem Sie mit einem Fingern nach links oder rechts wischen.

Position und Standardzoom wiederherstellen

Tippen Sie zweimal schnell hintereinander auf den Graph um die Position und den Zoom zurückzusetzen.

Klick auf Punkte für mehr Infos

Sie können auf die Punkte im Graph tippen um Infos zu den Werten an dieser Stelle zu bekommen.

Graph-Optionen

Oben rechts im Graphen befindet sich ein Menü, in dem Sie weitere Optionen für diesen Graphen, wie die Anzeige einer Trendlinie, wählen können.
